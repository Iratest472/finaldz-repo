Команды GIT

  • Команда git status для того, чтобы посмотреть какие изменения произошли в репозитории
  • Команда git commit -m "first commit для того, чтобы сделать коммит
  • Команда git push для того, чтобы внести изменения на удаленный репозиторий
  • Команда git log для того, чтобы вывести список всех наших коммитов.
